# Welcome to MDBlog

This is a sample blog post to demonstrate the MDBlog functionality.

## Features

- Markdown support
- Static site generation
- Simple configuration
- Local development server

## Getting Started

1. Put your markdown files in the `posts` directory
2. Run `./mdb` to generate the static site
3. Run `./mdb server` to start the development server

Happy blogging!
